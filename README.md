- Tensorflow
- Numpy
- Keras
- Scikit
- Android Studio


**TRASH SORTING RULES:**

| Bin's Color | Waste Type  |
| ----------- | ----------- |
| Blue        | Paper       |
| Green       | Glass       |
| Yellow      | Metal       |
| Black       | Mixed       |
| Brown       | Bio         |
